## Color Palette

Main brand colors:

- Orange: `FFAE44` brand accent
- Anthracite: `424A4D` secondary

Other colors:

- Blue: `46BDF4`
- Green: `93CF36`
- Purple: `8E3483`

[See all colors on the color wheel](https://color.adobe.com/Restorecommerce-color-theme-16094968).

## Typography

The following font faces are used:

- Logo: Lato
- Page: Muli
